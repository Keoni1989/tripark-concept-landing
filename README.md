# Tripark – Concept Landing Page

This repository contains an experimental landing page created to present the initial vision of Tripark.

Tripark was conceived as a platform designed to connect citizens and public administrations through reviews and information about public parks.

The purpose of this project was to explore:
- landing page design
- responsive layouts
- communicating a product idea
- rapid prototyping

## Tech

HTML
CSS
JavaScript

## Status

Archived concept.
The core Tripark application was never developed due to the complexity of the project and the need for a multidisciplinary team.
