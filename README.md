# Tripark – Concept Landing Page

A responsive landing page created to present the initial concept of Tripark, a platform designed to help citizens discover and review public parks.

## Why I built it

Before developing the full application, I wanted to explore how the idea could be presented through a landing page while practicing responsive design and product communication.

## Tech Stack

- HTML
- CSS
- JavaScript
- Vercel

## Status

Archived concept. The landing page was completed, while the full application remains an ongoing idea.
